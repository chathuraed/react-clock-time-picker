# react-time-picker

## Install

```bash
npm install --save react-time-picker
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-time-picker'
import 'react-time-picker/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
